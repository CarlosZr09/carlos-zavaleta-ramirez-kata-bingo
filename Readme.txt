# Abrir el archibo bingo.html para empezar a jugar
# En el archivo scripts_bingo podra encontrar todas las funciones que utilizamos para poder jugar :D
## la funcion get_numbers_random, la utilizamos para generar numberos de manera aleatoria que no se repitan y que sean del valor que corresponde por columna.
## la funcion generatingBingoCards, la utilizamos para generar la tarjeta de bingo en ella utilizamos la funcion get_numbers_random para poder pintar toda la tarjeta
## la funcion checkingBingoCards, la utilizamos para verificar que si se grito bingo verifiquemos que todos los numeros de su tarjeta fueron llamados.
## la funcion callingBingoNumbers, la utilizamos para llamar a un numero del 1-75 que no halla sido llamado previamente.
